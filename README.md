# ladirec_projetcafecafe_desambiguisation
Le projet « café-café » a pour but de désambiguïser deux termes, « café » le lieu et « café » la chose (breuvage, grain, industrie…), afin d’aider à repérer automatiquement les mentions d’emplacements géographiques dans les textes analysés par le Laboratoire d’analyse des discours et récits collectifs (LADIREC).
